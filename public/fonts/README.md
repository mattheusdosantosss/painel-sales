# Fontes self-hosted

Coloque aqui o arquivo da fonte de marca **Bruta Pro**.

- Formato ideal: **`.woff2`** → nomeie como **`Bruta.woff2`** e funciona direto
  (o `@font-face` em `app/globals.css` já aponta para `/fonts/Bruta.woff2`).
- Se tiver vários pesos (ex.: Regular + Bold), pode colocar todos e me avisar os
  nomes que eu declaro cada peso no `@font-face`.
- Se for `.otf`/`.ttf`, pode colocar assim mesmo — me diga o nome que eu ajusto o
  `src` (funciona no navegador, só é um pouco maior que woff2).

Arquivos de fonte não são versionados como texto; o Git trata como binário normalmente.
